# The Christman AI Project

Assistive AI systems designed by and for vulnerable populations.

We build software that protects human dignity: communication tools for the nonverbal, memory support for dementia care, crisis support for PTSD, and a sovereign development environment for family caregivers, educators, and clinicians.

Founded by Everett Christman. Built on Ollama, FastAPI, WebSockets, and local-first architecture. No lock-in, no cloud dependency, no fabricated cognition.

## Technical principles

1. **Offline-first** — core functions run on-device. Network is optional, not mandatory.
2. **Local models** — Ollama, llama.cpp, Qwen, local TTS/STT. No OpenAI lock-in.
3. **Speech-first** — built for nonverbal users and kids who can't or won't type.
4. **Fail loud** — systems report problems clearly; no silent hallucination.
5. **User owns the data** — no telemetry, no behavioral monetization.
